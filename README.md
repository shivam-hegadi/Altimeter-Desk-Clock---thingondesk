# Altimeter-Desk-Clock---thingondesk
An Altimeter Style Desk Clock for the thingondesk Hackclub program.


# Altimeter Desk Clock Design Journal

## Journal 1: Project Conception

I'm building an Altimeter Desk Clock that looks almost identical to one you'd find in a real aircraft. The idea is to capture that authentic instrument look while making something that actually works as a desktop clock.

## Journal 2: Base Sketch

Started on the base sketch for the housing. The center circle is where the clock face goes, and the surrounding area is for decorative detailing and texture. Not sure yet if I want to add texturing to that outer ring. I'll figure it out as I go.
<img width="669" height="668" alt="image4" src="https://github.com/user-attachments/assets/6fd835bb-b724-48cc-9f09-5a57dc5bd12e" />

## Journal 3: Main Body Extrusion

Extruded the main housing body to give it some depth and structure. Added a chamfered corner detail, which does two things: cuts down on print time by using less material, and makes it look more realistic like the actual instruments you see in aircraft.
<img width="541" height="548" alt="image5" src="https://github.com/user-attachments/assets/d57336c1-a89a-41d4-a48b-74910e32d8d0" />

## Journal 4: Clock Face Assembly

Designed the first piece of the clock face that sits inside the main body. The key part here is an integrated locking mechanism that holds the face in place without needing any pins or fasteners. This keeps everything seamless and clean looking. No visible hardware, which is what I wanted.
<img width="655" height="673" alt="image10" src="https://github.com/user-attachments/assets/63e47d2a-be05-4be7-bc42-f1fc72ca8385" />

## Journal 5: Face Retention and Next Steps

Added the cap retention feature that catches onto the main body. Now comes the tricky part: actually designing the clock face itself. This is where things get complicated because it needs precision, multiple intricate elements, and close attention to detail if I want it to look like a real altimeter.
<img width="798" height="454" alt="image12" src="https://github.com/user-attachments/assets/ed96c01d-1ecb-4778-8a40-f10cb32e0c5b" />

## Journal 6: Construction Line Layout

Added construction lines around the clock face at 45-degree intervals to use as placement guides. I'm planning to add 12 major tick marks and 24 smaller ones using these lines as reference. It's a tedious approach but it keeps everything evenly spaced.
<img width="751" height="758" alt="image8" src="https://github.com/user-attachments/assets/c4c55b3e-9c35-439f-b7e4-27e03dccd829" />

## Journal 7: Adding Tick Marks

Finished adding all 64 tick marks to the face. This was incredibly time-consuming. Way more repetitive than I expected with so many individual elements to create and position. Worth it though, because the detail really makes it look authentic.
<img width="607" height="573" alt="image11" src="https://github.com/user-attachments/assets/6304f5ff-cd03-4f0c-b936-de05f86f07da" />

## Journal 8: Face Details and Typography

Added text and numbers to the clock face to make it look like an actual instrument. Planning to print these in multiple colors so they stand out from the rest of the face. The color contrast should make the whole thing look more finished.
<img width="527" height="520" alt="image13" src="https://github.com/user-attachments/assets/d27f7ebc-d4fe-4854-b763-dd37ca6d076f" />

## Journal 9: Mechanism Fit Testing and Tolerance Adjustment

Imported a standard clock mechanism to see if it would fit inside my design. Turns out it barely fits. Almost no clearance at all. Had to add tolerance to the base design to give the mechanism some breathing room so it can function properly without binding up.
<img width="527" height="526" alt="image1" src="https://github.com/user-attachments/assets/ce7e0666-3d31-4e2c-b6fd-d838a8a47e07" />

## Journal 10: Final Design Check and Print Preparation

Did a final check on the design and exported the main body to the slicer for 3D printing. Tweaked the print settings to save filament while keeping the outer walls smooth and clean. Got the parameters dialed in to balance efficiency with quality.
<img width="1490" height="986" alt="image6" src="https://github.com/user-attachments/assets/4481e707-0d34-49c7-ad1c-f6631f86b70c" />

## Journal 11: Detailing, Rendering, and Multicolor Printing

Added some fine details to the face to make it look more realistic. Used my free Autodesk Education cloud credits to render the design so I could see what it'd look like before printing. After reviewing the renders, I sent the face to the printer with multicolor settings to create contrast between the different elements.
<img width="715" height="436" alt="image3" src="https://github.com/user-attachments/assets/e432f7b7-3a30-4b32-bd93-f65780086972" />
<img width="1999" height="1546" alt="image2" src="https://github.com/user-attachments/assets/8688cde0-b684-4023-bd8d-b911362c8c73" />

## Journal 12: Clock Hand Design

Designed both the hour and minute hands to match the style of aircraft altimeter hands. Kept the proportions realistic but scaled appropriately for a desk clock. Once I finalize the design, I'll send them to print.
<img width="477" height="322" alt="image9" src="https://github.com/user-attachments/assets/99eb3a01-3214-4eed-bfaf-d5e5dea5f346" />

## Journal 13: Final Assembly and Completion

Got both hands installed and the clock is done. This part required some reprinting and tweaking to get the fit right, but after a few iterations everything came together. The final product is complete and actually works.
